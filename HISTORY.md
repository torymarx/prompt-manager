# HISTORY

## [v3.3.0] - 2026-03-26
### Added
- **북마크(링크) 개별 등록 기능**: 어떤 폴더나 뷰에서도 '새 프롬프트'와 '새 링크'를 선택하여 등록할 수 있는 드롭다운 메뉴 도입.
- **범용 에디터 지원**: 프롬프트 리스트 뷰에서도 간결한 북마크 전용 입력창(`WebsiteEditor`)을 호출할 수 있도록 개선하여 북마크 등록 편의성 증대.

### Fixed
- **'북마크' 폴더 무한 생성 해결**: 웹사이트 폴더가 없을 때 시스템이 강제로 '북마크' 폴더를 생성하던 로직을 제거하여, 사용자가 폴더를 완전히 삭제할 수 있도록 수정.
- **전체 게시물 우선순위 정렬**: 전체 보기에서 인물/그림 폴더 게시물이 상단에, 지침프롬프트/북마크/유튜브 폴더 게시물이 하단에 표시되도록 정렬 로직 적용.

## [v3.2.0] - 2026-03-22
### Added
- **댓글 갯수 표시**: 대시보드 프롬프트 카드에 실시간 댓글 갯수 배지 추가.

## [v3.1.1] - 2026-03-22
### Fixed
- **댓글 삭제 동기화**: 댓글 삭제 시 화면이 자동으로 갱신되지 않던 문제 해결 (낙관적 업데이트 및 실시간 구독 로직 개선).

## [v3.1.0] - 2026-03-22
### Added
- **댓글 시스템**: 프롬프트 상세 및 공유 페이지에 댓글 기능 도입.
- **감정 표현(Reactions)**: 댓글에 다양한 이모지 반응 기능 추가.
- **이미지 업로드**: 댓글에 참고 이미지를 직접 업로드할 수 있는 기능.
- **실시간 동기화**: Supabase Realtime을 통한 실시간 댓글 업데이트.

## [v3.0.0] - 2026-03-22
### Added
- **프롬프트 내용 미리보기**: 리스트 뷰에서 프롬프트 제목 아래에 내용의 첫 1~2줄(Snippet)이 표시되도록 개선하여 클릭 전 정보를 확인할 수 있게 함.

### Changed
- **검색창 반응형 최적화**: 모바일 및 태블릿 환경에서 검색창 너비가 유연하게 조절되도록 레이아웃 엔진 고도화.
- **브레드크럼(Breadcrumb) 개선**: 폴더 깊이가 깊을 때 모바일에서 가로 스크롤이 가능하도록 수정하고, 텍스트 생략(Truncation) 로직을 정교화함.
- **사이드바 디자인 강화**: 선택된 폴더의 시각적 대비(초점 효과)를 높여 현재 위치를 더 명확하게 인지할 수 있도록 개선.
- **버전 통합**: 프로젝트 전체 버전을 `v3.0.0`으로 통합 관리 시작.
